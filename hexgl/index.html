Play HexGL with xLabs!


HexGL is a HTML5 futuristic game built by Thibaut Despoulain.

xLabs is a lightweight head pose and gaze tracking application through your webcam.

I have built a controller using xLabs that allows you to control your ship via head-movement.

In order to play with head-movement, you need to install xLabs first.

What is xLabs?

xLabs is a chrome extension for head pose and eye gaze tracking. It use the webcam; no other external devices are needed. You can find some demos here

View details »

What is HexGL?

HexGL is a futuristic, fast-paced racing game built by Thibaut Despoulain using HTML5, Javascript and WebGL. It was built with three.js, a 3D library built on top of WebGL. And the metalic road texture was built by Nobiax

View details »

How to play with xLabs?

Install the xLabs chrome extension, restart your browser. At the launching page, use the default xLabs controller and start the game. Then you can control your ship by slightly tilting your head.
